# App_Web_nodex

## Cloner :   

```
$ git clone https://github.com/RamzyK/App_Web_node/tree/master
```

  ## Installer :   

```
$ npm install
```

  ## Éxecuter :   

```
$ DEBUG="node-app:server" npm start
```

  ## Déployer :   

```bash
$ heroku login
$ heroku create
$ git push heroku master
$ heroku open
```

## Test :

```
Lancer le serveur:
	$ npm start
Lancer le test:
	$ npm test
```
